# Forced Alignment Pipeline

## Repository Contents
* `Forced_Alignment_Pipeline.ipynb`: Complete execution log. Contains the MFA setup code (Task 1-4) and the Neural Alignment fallback (Task 5).
* `TextGrid_Outputs/`: The final aligned files.
* `Report.pdf`: Analysis of the alignment and OOV handling.

## How to Run
Open `Forced_Alignment_Pipeline.ipynb` in Google Colab to replicate the experiment.
